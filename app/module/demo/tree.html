树使用说明

描述：

树提供了大部分树中常用的功能。

API说明：

参数 默认值 说明
tree-data 标签中的数据属性，当URL存在时tree-data中的值无效，二者只要选择一种，即通过tree-data直接设置值或使用URL从后台获取，URL获得方式级别大于tree-data方式
options 标签中的参数属性
options.url 空 AJAX请求后台服务器地址，当URL存在时tree-data中的值无效，二者只要选择一种，即通过tree-data直接设置值或使用URL从后台获取，URL获得方式级别大于tree-data方式
options.method POST AJAX请求类型，使用tree-data设置数据时无效
options.params 空 AJAX请求参数，使用tree-data设置数据时无效
options.asyncTree false 异步树，使用tree-data设置数据时无效
options.canChecked false 使用选择框
options.cascadeChecked false 级联选择
options.singleChecked false 单选，canChecked为true时有效果
options.hideParentCheckBox false 隐藏父选择框，canChecked为true时有效果
options.singleCheckedNodeId 空 单选，根据id节点选择节点，singleChecked与canChecked为true时有效果
options.animatedCls 空 动画样式
options.onCheckboxChanged function(node,$event){} 选择框改变事件
options.onNodeClicked function(node,$event){} 节点单击事件
instance 标签中的实例属性
instance.getCheckedData 获得所有数据
instance.getAllData 获得选中的数据
instance.setCheckedNodeById 根据id选择一个节点
instance.setCheckedNodeByIds 根据ids选择节点
instance.checkedAllByState 根据状态选中节点
