使用说明

描述：

提供最基本的分页功能，如上一页、下一页、第一页、最后一页与刷新。

使用：

							
html部分
参考示例


JS部分

//分页配置
$scope.pagerConfig = {
	total: 30, //总条数
	pageList: [15, 25, 35] //每页显示条数
}

//分页改变事件
$scope.pagerChange = function(type, pageParam) {
	console.info(type); //类型，分别为：fristPage(第一页)、previousPage(上一页)、refresh(刷)、nextPage(下一页)、lastPage(最后一页)
	console.info(pageParam); //参数，分别为：pageNumber(当前页)、pageSize(当前页条数)、pageTotal(总页数)、total(总条数)

}

API说明：

参数 默认值 说明
setOptions 设置配置
setOptions.total 0 总条数
setOptions.pageList [10, 20, 30, 50, 100, 150, 200] 每页条数
onPageChanged 分页改变事件，事件传入type, pageParam两个参数， type，分别为：fristPage(第一页)、previousPage(上一页)、refresh(刷)、nextPage(下一页)、lastPage(最后一页)等值， pageParam，分别为：pageNumber(当前页)、pageSize(当前页条数)、pageTotal(总页数)、total(总条数)等参数。
